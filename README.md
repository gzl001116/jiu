# 我的网站

这是一个使用 Fluid Player 构建的视频博客网站。

## 内容

- index.html - 主页
- video-blog.html - 视频博客页面
- vlog.html - Vlog 页面
- about.html - 关于页面

## 部署

网站通过 GitHub Actions 自动部署到 GitHub Pages。
