DPlayer 视频播放器演示

关于 DPlayer

DPlayer 是一个现代化的 HTML5 视频播放器，支持以下功能：

  • 弹幕支持
  • 多种视频格式支持 (MP4, HLS, FLV 等)
  • 响应式设计
  • 键盘快捷键
  • 多种字幕格式支持
  • 清晰度切换
  • 截图功能
  • 画中画模式
  • 倍速播放

使用方法

在您的项目中引入 DPlayer：

// 安装
npm install dplayer

// 使用
import DPlayer from 'dplayer';

const dp = new DPlayer({
    container: document.getElementById('dplayer'),
    video: {
        url: 'demo.mp4',
        pic: 'demo.jpg',
        thumbnails: 'thumbnails.jpg'
    }
});
